# 📅 gitMonkCal - Full Business Calendar 2 iOS Clone

High-density iOS calendar matching **every BC2 feature**: Month/Week(1-14)/Day(Weather+Tasks+Birthdays)/Agenda/Tasks/Year(Heatmap), templates, recurrence, 5 alarms, cancel events, drag/delete/share/print/voice(placeholder), 22 themes, calendar colors, attachments(notes/subtasks), search.

**Tabs**: Month | Week | Day | Agenda | Tasks | Year | More(Settings/Themes).

**Build**: `tuist generate` → Xcode → Any iOS Device → SideStore IPA.

iOS Native: SwiftUI/EventKit/WidgetKit/WeatherKit. MIT License.
